Raspar dados de sites protegidos pelo CloudFlare

RecaptchaV2, fácil comparado a proteção anti-bot do ClodFlare. Enfrentei desafios em que tive que extrair dados em sites protegidos pela CloudFlare e muitas vezes preciva fazer login para acessar o conteúdo alvo do scraping. Eles são fortes, mas não existe site que não possa ser extraidos os dados por bots. Todos eles sempre tem uma vulneralbilidade, uma API aberta, um Curl que traz todo conteúdo html da página, um tráfego de rede desprotegido, um ajax que pode ser acessado...
